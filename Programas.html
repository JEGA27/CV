Programas

En esta página desscribo mis programas favoritos que he hecho a lo largo de mi carrera en diferentes lenguajes de programación.

Funko

Para este programa se desarrolló en python un prototipo de página web de los productos de Funko agregando nuevos métodos y filtros de bússqueda de sus productos, además de una pequeña base de datos con una cierta cantidad de productos de diferentes categorías.

App IOS

Mediante la herramienta xcode y el lenguaje swift se desarrolló una aplicación para dispositivos IOS para una clínica de tanatología, en la cual los usuarios llevan un control de los servicios, las sesiones y citas que se llevan a cabo en la clínica, así como también se puede solicitar el servicio de ambulancia y subir los expedientes de los pacientes.

Streaming de Video

Mediante programación orientada a objetos con el lenguaje C++ se desarrolló un programa que imita el funcionamiento de las plataformas de video, contanto con diferentes clases como "Película" o "Serie" almacenando cada una videos e información de ellos como el rating, permitiendo al usuario buscar videos ya sea mediante el título de la seria o película, así como ver su rating y más información de cada video.

Videojuego

Mediante el motor de videojuegos Unity y código en C# se desarrolló un videojuego en 3D con temática de escape para impulsar a los usuarios a estudiar carreras de ingeniería. El videojuego se basa en una inundación en una ciudad, por lo que el personaje se encuentra atrapado en el edificio así que tiene que resolver retos relacionados con programación para ir subiendo de piso hasta lograr llegar a la asotea y ser rescatado escapando de la inundación

Página Web

Para la elaboración del videojuego se creó también una página Web en la cual las estadísticas del videojuego de cada usuario se subían para mostrarlas así como para generar tablas de ranking. Para hacer esto posible, se utilizó SQL para mantener el flujo de los datos desde el videojuego hacia la base de datos y después a la página Web, de esta manera el sitio automáticamente recuperaba los datos de cada sesión del videojuego.
